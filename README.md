# Drishti_ai_powered_drug_peddlig_detection_and_aalysis_system
DRISHTI — AI-powered drug trafficking detection system. Collects live Reddit data, detects suspicious codewords, classifies users as Seller/Buyer/Normal using Random Forest + TF-IDF, clusters behaviour with KMeans, and visualises intelligence on a real-time Streamlit dashboard mapped to Indian cities.
